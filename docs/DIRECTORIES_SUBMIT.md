# AI 目录提交清单

用于 **AILingo**（https://ailingo.vercel.app/）在 1 个月内上架多个 AI 应用目录，配合 `ONE_MONTH_LAUNCH.md` 冲榜。

---

## 1. Product Hunt

| 项目 | 说明 |
|------|------|
| **入口** | https://www.producthunt.com → 右上角 **Submit** → **New Product** |
| **必填** | Product URL、Name、Tagline、Topics、Pricing、Thumbnail 240×240、Gallery 至少 2 张 1270×760、Status |
| **可选** | Video（仅 YouTube 完整链接）、Download links |
| **文案** | 见 `ONE_MONTH_LAUNCH.md` 第八节 |

---

## 2. FuturePedia

| 项目 | 说明 |
|------|------|
| **入口** | https://www.futurepedia.io/submit-tool |
| **注意** | 当前为付费上架：Basic Listing $247（约 7 天内上线）、Verified $497（约 2 个工作日 + 认证标 +  newsletter 等）。若预算有限可优先做 PH + 下面免费目录。 |
| **必填** | 工具名称、URL、描述、分类（选 Education / AI Tools）、Logo、截图等（以页面表单项为准） |

---

## 3. There's An AI For That (TAAFT)

| 项目 | 说明 |
|------|------|
| **入口** | https://theresanaiforthat.com/submit/ 或 https://taaft.com/submit |
| **分类** | 可选 All tools / Agents / GPTs 等，AILingo 选 Web 或 Education 类 |
| **必填** | 产品 URL、名称、描述、分类、图标/图（以页面为准） |
| **福利** | 新工具首发有机会获 PPC 奖金等，见站内说明 |

---

## 4. 其他免费/低成本目录（任选 1–2 个）

| 目录 | 入口 | 备注 |
|------|------|------|
| **AI Hunt** | 搜索 "AI Hunt submit" 或 "AI Hunt add tool" 获取当前提交页 | 多为表单：URL、名称、描述、分类 |
| **ToolScout / AI Tool Guru** | 各站 "Submit" / "Add tool" 页面 | 填产品链接与描述即可 |
| **SaaS Hub / BetaList** | 若定位为 EdTech SaaS | 适合「即将上线」或「已上线」产品 |

提交前请确认该目录仍接受新工具且分类中有 **Education** 或 **AI Learning**。

---

## 提交前通用准备

- **Product URL**：https://ailingo.vercel.app/
- **Name**：AILingo
- **Tagline**（英）：Learn AI & LLM with AI-generated micro-lessons — Duolingo-style.
- **Short description**（约 1–2 句）：Turn papers, URLs, and topics into bite-sized lessons. Concept cards, code fill-in, quizzes, progress tracking. Mobile-first.
- **分类**：Education, Artificial Intelligence, Developer Tools, EdTech
- **Pricing**：Free (with optional Pro)

---

## 执行顺序建议

1. **Week 1**：准备好主图 240×240、Gallery 2 张 1270×760、Tagline/Description；注册/登录 PH、TAAFT。
2. **Week 2**：先发 **Product Hunt**；同周提交 **There's An AI For That**（免费）。
3. **Week 2–3**：再选 1 个免费目录（如 AI Hunt）提交；若预算允许再考虑 FuturePedia。
4. 每提交一处，在表格里打勾并记下「提交日期 + 链接」，便于后续更新与复盘。
